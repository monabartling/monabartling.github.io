ONGOING RESEARCH

Project title: Novel approaches for context-based map design adaptation

In our mobile, digital society, mobile maps are increasingly used in different aspects of our lives, leading to an increase in map use situations (e.g., differences in users’ spatial or digital skills, temporal or environmental distractions, such as noise, direct sunlight, rain, etc.). Adjusting the map design towards these differences (e.g., adjusting the symbology to respond to light conditions, reduce interactivity complexity to respond to users’ cognitive constraints, etc.) supports map users to use mobile maps in these diverse map use situations. The evaluation of differences in map use situations is therefore relevant for designing suitable maps, and for defining how map designs should vary or adjust between map use situations.

This project expands on previous research efforts and seeks to evaluate novel approaches for map design transferability and map design adaptation to respond to varying map use situations. The project particularly focuses on techniques to model map use situations (i.e., map use context factors) and analyzes methods to dynamically evaluate the usability of the transferred and adapted map design choices. The project further aims to evaluate ways to identify what are relevant aspects of map use situations that define what constitute differences between map use situations.
